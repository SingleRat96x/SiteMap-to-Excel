Privacy Policy

Last updated: January 2025

Introduction

This privacy policy explains how the Sitemap to XLSX Extension ("we", "our", or "the extension") handles your information.

Current Data Collection

The extension currently collects and processes:

  • Website content (sitemap URLs) - Required for core functionality
  • Temporary local storage data for extension settings

All current data processing is done locally in your browser, and no data is sent to external servers except for fetching sitemaps from the URLs you provide.

Future Data Collection

In future versions, we plan to implement Google AdSense advertising, which will involve:

  • Location data (IP address, region) - Required for relevant ad delivery
  • User activity (ad interactions) - Required for ad performance tracking
  • Device information - Required for ad delivery optimization

Users will be notified before these changes take effect and will have options to manage their ad preferences.

Data We DO NOT Collect

The extension does not and will not collect:

  • Personally identifiable information
  • Health information
  • Financial and payment information
  • Authentication information
  • Personal communications
  • Web browsing history
  • Keystroke logging

Data Usage and Sharing

  • Current data is processed locally and not shared with third parties
  • Future ad-related data will be shared only with Google AdSense
  • We never sell your data to third parties
  • Data is only used for extension functionality and ad delivery

Your Rights and Controls

You have the right to:

  • Clear all stored data using the extension's reset function
  • Manage ad preferences (in future versions)
  • Uninstall the extension at any time

Security

We implement security measures to protect your data:

  • All network requests use HTTPS
  • Content Security Policy (CSP) to prevent unauthorized code execution
  • Local data storage with browser security controls

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes through extension updates.

Contact

For any questions about this privacy policy or our privacy practices, please contact us through the Chrome Web Store support channel.
